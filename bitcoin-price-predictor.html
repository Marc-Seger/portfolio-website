Bitcoin Price Prediction

Work in Progress

Project Overview

This project focuses on developing a machine learning-based Bitcoin price prediction tool. Using Random Forest and GRU models, the system forecasts short-term and long-term Bitcoin prices by leveraging technical indicators, macroeconomic data, and blockchain metrics.

Current Status

The prediction logic is fully implemented, with ongoing improvements to model accuracy and feature simulation. A Streamlit app is under development to provide interactive forecasts based on user-selected dates.
