Bitcoin Price Predictor

Forecasting Bitcoin using ML models: GRU & Random Forest

🔍 Overview

This application uses machine learning models (Random Forest and GRU) to forecast Bitcoin prices over different time horizons.

Short-term predictions are made using tree-based models like Random Forest, while long-term trends are captured using deep learning (GRU). The models leverage a rich feature set including:

  • 📈 Technical indicators (MACD, EMA, Bollinger Bands, etc.)
  • 💰 On-chain data (active addresses, exchange flows)
  • 🧠 Sentiment signals (Google Trends, Fear & Greed Index)
  • 🌍 Macroeconomic data (S&P 500, Gold, DXY, CPI)
  • 🔁 Lagged prices & rolling averages
