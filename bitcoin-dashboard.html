📊 Bitcoin & Assets Dashboard

🔍 Project Overview

This interactive dashboard tracks Bitcoin alongside key financial assets such as the S&P 500, Nasdaq, Gold, and macroeconomic indicators. Built using Python and Streamlit, it allows users to explore trends, compare indicators, and monitor market signals in real time.

✨ Features

  • Real-time Bitcoin price and market KPIs
  • Dynamic line/candlestick charts with technical overlays
  • Signals & sentiment summaries (MACD, VWAP, RSI, etc.)
  • Spot ETF flow analysis (daily & cumulative)
  • Fear & Greed Index + Google Trends visualization
